# Triade Web

Solução empresarial para gerenciamento de clientes, funcionários e agendamentos.

## Tecnologias utilizadas:
  - NPM: v.6.4.1
  - Vue CLI: v.3.2.1
  - Vue.js: v.2.6.10
  - Vuex: v.3.0.1
  - Vue Router: v.3.0.3
  - Vue Resource: v.1.5.1


### Instalando as dependências e rodando o projeto:
```
npm install && npm run serve
```

configuração de acesso a API `src/http/index.js`
